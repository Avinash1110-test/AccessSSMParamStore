# AccessSSMParamStore
#### In this project we'll learn how to get parameter value from AWS SSM Paramter Store.

## Technologies:
* Java 11
* Spring Boot 2.7.5-SNAPSHOT
* Maven 3.8.6
* AWS
## Tools:
* Intellij
* Git
* Postman
##### Note:
We need AWS account with Systems Manger access to run and test this application.
